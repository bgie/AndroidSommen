Android Sommen
=============

Dit is een app voor Android 2.1 en hoger. Deze app is een reken app voor kinderen van 6-12 jaar, ouders en "grote kinderen" zijn ook inbegrepen.

FEATURES
-------

 * Eenvoudig en leuk
 * Mogelijke oefeningen: van 0 tot oneindig
 * Zowel + - * / en eender welke mix
 * Topscores
 * Multi user save games
 * en nog veel meer!

Ontstaan
--------

Dit is een opdracht van de JAVA lessen in SYNTRA Genk. Deze klasopdracht wordt via git klassikaal gemaakt.
